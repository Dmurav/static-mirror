План разработки

1. создаём docker image на основе Ubuntu, устанавливаем необходимые пакеты и настраиваем crontab, отправляем image в dockerhub
2. на базе подготовленного образа создаём Dockerfile, где прописываем команды при запуске контейнера
3. для автоматизации запуска подготавливаем docker-compose.yaml, который автоматически создаст контейнер на основе указанного образа и смонтирует директорию для хранения данных загруженного сайта
4. подготавливаем Makefile в котором прописываем docker-compose команды
5. подготавливаем README.md
6. выгружаем локальную версию git репозитория на github
